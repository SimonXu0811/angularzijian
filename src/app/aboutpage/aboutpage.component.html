How did I develop this website?

The tools I use

Docker
This website is built as a docker imagine. And the dockerfile is first pull node image to run npm install. And then, pull nginx to start website server. Because this website is a SPA(Single Page Application), I reconfingure the nginx to handle routing 404.
AWS-EC2
The server is host on Amazon Linux 2 AMI. First, I configure and start EC2 instance. And then connect to the virtual machine with SSH. Second, install all tools such as git and docker with yum. Third, pull the project with git, run dockerfile. Last, I run the container on the server, so I can get the access with public DNS(IPv4).
AWS-Route 53
To change the DNS, I use Route 53 to register a domain name. After the DNS registered successfully, I added the hosted zones and record set. So now I can access my website with my domain name.
